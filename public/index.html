About

Hello! My name's Casey, and I'm a software developer currently based in Denver.

I'm primarily focused on frontend work, but my interests span into the backend. I love all things Javascript.

I strive to build technology that is:

Useful

Knowing how to code is like having magical powers: I can see a need and pull a thing out of nothingness to meet that need. I feel most fulfilled when I use these powers to build technology that helps people, making their lives easier.

Responsive

Responsiveness means more than just screen-sizes; it functioning seemlessly across browsers and operating systems as well. No one wants to be trapped in front of a desktop machine all day. (Actually, some of us don’t mind it so much…)

Accesible

Accomodating people with different abilities is not an extra feature, it is a fundamental responsibility. For many, an inaccesible site is a broken site. Developers must build accesible products if the internet is to live up to its democratizing potential.

Shoot The Breeze

A realtime chat app using React and Firebase.

Sit around and chat beside the firebase. Built with React and Auth0, leveraging Firebase’s live updating feature, this app allows entry to the chat room through the user’s Google account. It’s mobile responsive, has a twitter-like 140 charater limit, and features a panel of all signed-in users.

  • Code: Github
  • Deployed: Hit me up in the chatroom here!

This Just In!

Choose between 60+ news feeds to create your perfect newspaper.

A headline aggregator, organized like a personalized newspaper. Built with React, Redux, and Firebase. The user can log in and select from any of 60+ different news feeds, organized by ‘page’ (sports, business, world, etc.). Customized news pages are displayed, and the selections persist between sessions. A solo project.

  • Code: Github
  • Deployed: Create your paper here!

Blanched Almonoid

A classic arcade game in pure JS, named after a CSS color.

A version of the classic Breakout-style game, named in tribute to FFEBCD, aka ‘Blanched-Almond’. It is built purely in JavaScript, in a very object-oriented prototype-based architecture. Styled after a 50’s cookbook theme. The game allows the player to choose any of 10 different levels to play. Clicking the ‘Start Blanchin!’ button gives the player a countdown timer before the almond is launched. Gameplay continues until the player clears all the bricks or loses the almond 3 times in any given level. At any time a player is able to change the level using the buttons in the lower left corner of the screen.

  • Code: Github
  • Deployed: Break some bricks here!

Re:fuel

Find nearby fueling stations for your alternatively-fueled vehicle.

A project for iOS, built with React Native. The app consumes an API to find nearby alternative fueling stations (electric, biodiesel, etc.) that fit the user’s needs based on selected preferences. Uses geolocation to calculate the closest stations, and provides detailed information about each station.

  • Code: Github
  • Deployed: This app not yet deployed to the iOS App store.

A full MVC todo app with Ember.

The UI implements a classic master/detail view, allowing users to easily create, read, update, and destroy their reminders. Features instant filtering and caution notices when data is unsaved.
